CÁCH CÀI NHANH:

1. Upload toàn bộ file/folder trong ZIP lên repo:
   Vaynhanh247vn

2. Tạo Google Sheet:
   VayNhanh247_Leads

3. Google Sheet -> Extensions -> Apps Script

4. Copy toàn bộ:
   google-apps-script/Code.gs

5. Deploy:
   Deploy -> New deployment -> Web app
   Execute as: Me
   Who has access: Anyone

6. Copy Web App URL.

7. Mở:
   assets/config.js

8. Dán URL vào:
   GAS_URL: 'https://script.google.com/macros/s/.../exec'

9. Upload lại riêng:
   assets/config.js

10. Test form.
