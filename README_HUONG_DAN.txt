- CSS/JS đủ

Cài:
1. Giải nén ZIP.
2. Upload toàn bộ file/thư mục lên GitHub repo `taichinh247vn`.
3. Commit changes.
4. Chờ GitHub Pages build 1-2 phút.
5. Mở: https://thanhbn2910.github.io/taichinh247vn/?v=161
6. Blog: https://thanhbn2910.github.io/taichinh247vn/posts/
7. Admin: https://thanhbn2910.github.io/taichinh247vn/admin.html

Gắn Google Sheet:
1. Tạo Google Sheet.
2. Extensions → Apps Script.
3. Dán file google-apps-script/Code.gs.
4. Deploy Web App, quyền Anyone.
5. Copy URL, dán vào assets/app.js dòng GAS_URL:''.
6. Upload lại assets/app.js lên GitHub.
