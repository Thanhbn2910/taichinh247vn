V18.2 — KÍCH HOẠT GOOGLE SHEET THẬT + GMAIL THẬT

B1 — Tạo Google Sheet thật
Tên file:
VayNhanh247_Leads

Mở:
Extensions → Apps Script

B2 — Dán code
Trong gói web có file:
google-apps-script/Code.gs

Mở file đó → copy toàn bộ → dán vào Apps Script → Save.

B3 — Deploy Web App
Bấm:
Deploy → New deployment

Chọn:
Type: Web app
Execute as: Me
Who has access: Anyone

Bấm:
Deploy → Authorize → đăng nhập Gmail.

Copy link dạng:
https://script.google.com/macros/s/AKfycbxxxx/exec

B4 — Kích hoạt web
Mở file:
assets/config.js

Đang là:
window.VAYNHANH247_CONFIG = {
  GAS_URL: ''
};

Đổi thành:
window.VAYNHANH247_CONFIG = {
  GAS_URL: 'https://script.google.com/macros/s/AKfycbxxxx/exec'
};

B5 — Upload lại GitHub
Upload file:
assets/config.js

Commit:
Activate Google Sheet + Gmail
