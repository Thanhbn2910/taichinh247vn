V18.1 Setup

Hướng dẫn kích hoạt Google Sheet + Gmail Lead thật

Bước 1 — Tạo Google Sheet

Tạo Google Sheet mới tên: VayNhanh247_Leads

Bước 2 — Apps Script

Trong Google Sheet chọn: Extensions → Apps Script. Xóa code mặc định, copy toàn bộ file google-apps-script/Code.gs vào.

Bước 3 — Deploy Web App

Chọn: Deploy → New deployment → Web app

  • Execute as: Me
  • Who has access: Anyone

Copy link dạng: https://script.google.com/macros/s/.../exec

Bước 4 — Dán link vào web

Mở file assets/config.js và dán link vào dòng:

GAS_URL: 'https://script.google.com/macros/s/.../exec'

Bước 5 — Upload lại GitHub

Upload lại file assets/config.js lên repo Vaynhanh247vn, commit và test form.
