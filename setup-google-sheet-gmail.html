V18.2 Setup Complete

Kích hoạt Google Sheet thật + Gmail thật

B1 — Tạo Google Sheet

Tạo file: VayNhanh247_Leads

Mở: Extensions → Apps Script

B2 — Dán Code.gs

Mở file google-apps-script/Code.gs, copy toàn bộ và dán vào Apps Script.

B3 — Deploy Web App

Deploy → New deployment → Web app

  • Execute as: Me
  • Who has access: Anyone

Copy Web App URL dạng https://script.google.com/macros/s/.../exec

B4 — Dán URL vào assets/config.js

window.VAYNHANH247_CONFIG = {
  GAS_URL: 'https://script.google.com/macros/s/AKfycbxxxx/exec'
};

B5 — Upload lại assets/config.js

Commit với nội dung: Activate Google Sheet + Gmail

B6 — Test

Test trang chủ Mở CRM
