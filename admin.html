Kết nối Google Sheet DB

  1. Mở file google-apps-script/Code.gs.
  2. Tạo Google Sheet mới, đặt tên sheet là Leads.
  3. Vào Extensions → Apps Script → dán Code.gs.
  4. Deploy → New deployment → Web app.
  5. Who has access: Anyone.
  6. Copy Web App URL và dán vào assets/app.js tại dòng GAS_URL:''.

Sau bước này, form ngoài website và admin sẽ dùng dữ liệu Google Sheet thật.
